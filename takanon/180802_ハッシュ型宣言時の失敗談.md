# ハッシュの文字列キーとシンボルキーで混乱したお話

## 概要

文字列キーのハッシュを用意しようとした  

```
> hoge = {'piyo': 1}
> hoge['piyo']
=> nil
```

あれ...？

## 問題点

```
hoge = {'piyo': 1}
=> hoge = {:piyo=>1}
```

シンボルなんですねこれ。


## おとなしくロケットで

```
> hoge = {'piyo' => 1}
=> hoge = {"piyo"=>1}
```

普段から生成されるものを意識していないから  
こうやって無駄な時間を使うんだと反省
